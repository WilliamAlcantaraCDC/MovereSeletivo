O projeto está captando o nome do cliente, data e o valor da venda, salvando os dados no banco de dados e possui uma opção de exibir a maior venda;
Página inicial chama-se index, o nome do banco chama-se base, o nome da tabela chama-se clientes;
Possui 3 botões, enviar, cancelar e mostrar a venda de maior valor;
Foi utilizado a linguagem PHP e as ferramentas Notepad++ e Wampserver;
